## 설치 방법
1. TPM 설치: git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
2. tmux 내에서: prefix + I (플러그인 설치)
3. prefix + U (플러그인 업데이트)
4. prefix + alt + u (플러그인 제거)
